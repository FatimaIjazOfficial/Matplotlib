# 📊 Matplotlib Basics

## 📌 Overview

This project covers the **fundamentals of Matplotlib** for data visualization using simple datasets.  
It includes different types of plots used in Data Science and EDA.

### 📊 What’s Included:
1. Blank figure setup  
2. Line plot (Sales trend)  
3. Scatter plot (Temperature vs Ice cream sales)  
4. Bar chart (Monthly sales comparison)  
5. Histogram (Age distribution)  
6. Combined 2x2 subplot visualization  

---

## 📦 Library Used

- Matplotlib → for plotting graphs and visualizations

---

## 📄 1. Blank Figure

### 📌 What This Does
Creates an empty figure with:
- Title
- X-axis label
- Y-axis label

### 🎯 Learning Outcome
- Basic figure creation
- Axis labeling
- Saving plots as images

---

## 📈 2. Line Plot (Sales Trend)

### 📌 What This Shows
A weekly sales trend visualization using a line plot.

### 🎯 Learning Outcome
- Trend analysis over time
- Line plotting with markers
- Basic data visualization of time-series data

---

## 📊 3. Scatter Plot (Temperature vs Ice Cream Sales)

### 📌 What This Shows
Relationship between temperature and ice cream sales.

### 🎯 Learning Outcome
- Understanding correlation visually
- Scatter plot usage for relationship analysis
- Pattern identification

---

## 📊 4. Bar Chart (Monthly Sales Comparison)

### 📌 What This Shows
Comparison of sales across different months.

### 🎯 Learning Outcome
- Categorical data visualization
- Comparing values across groups
- Bar chart interpretation

---

## 📉 5. Histogram (Age Distribution)

### 📌 What This Shows
Distribution of age values grouped into bins.

### 🎯 Learning Outcome
- Frequency distribution understanding
- Data grouping using bins
- Statistical visualization basics

---

## 📊 6. Combined Subplots (2x2 Grid)

### 📌 What This Shows
All plots combined into a single figure:
- Line plot
- Scatter plot
- Bar chart
- Histogram

### 🎯 Learning Outcome
- Multi-plot visualization
- Subplot grid system (2x2)
- Efficient dashboard-style plotting

---

## 📁 Output Files Generated

After execution, the following images are saved:

- Blank figure image
- Line plot (sales trend)
- Scatter plot (temperature vs sales)
- Bar chart (monthly sales)
- Histogram (age distribution)
- Combined subplot visualization

- Matplotlib basics
- Data visualization fundamentals

- Subplot grid layout
- Real-world data representation
