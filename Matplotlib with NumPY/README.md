# 📊 Matplotlib Data Visualization with NumPy

## 📌 Overview

This project demonstrates basic data visualization techniques using **Matplotlib** and **NumPy**.  
It focuses on generating synthetic data and visualizing it using different types of plots.

The goal is to practice:
- Data visualization
- Plot customization
- Subplots handling
- Statistical data representation

---

## 📦 Libraries Used

- NumPy → for generating random and mathematical data
- Matplotlib → for creating visual plots

---

## 📊 1. Histogram: Group Comparison

### 📌 What This Does
Two normally distributed datasets are generated:
- Group A (mean = 70, std = 10)
- Group B (mean = 80, std = 12)

These are compared using histograms to visualize distribution differences.

### 🎯 Key Learning
- Data distribution comparison
- Density-based histogram plotting
- Overlaying multiple histograms

---

## 📊 2. Subplots Overview

### 📌 What This Does
A single figure is divided into 4 sections showing:
- Line plot (trend visualization)
- Scatter plot (data relationship)
- Histogram (distribution)
- Bar chart (categorical comparison)

### 🎯 Key Learning
- Multi-plot figure creation
- subplot grid system
- Handling multiple visualizations in one canvas

---

## 📈 3. Customized Sine Wave Plot

### 📌 What This Does
A sine wave is plotted with enhanced customization:
- Custom axis ticks
- Axis labels
- Legend positioning
- Rotated x-axis labels
- Saved as an image file

### 🎯 Key Learning
- Plot styling and formatting
- Axis customization
- Figure saving for reports/projects

---

## 📁 Output Files Generated

After execution, the following images are saved:

- Histogram comparison plot
- Subplot grid visualization
- Customized sine wave plot

- Statistical visualization

- Figure customization and export
